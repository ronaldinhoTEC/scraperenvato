Documentación de Scripts

1. urls.py

Este script obtiene todas las URLs desde https://elements.envato.com/es/wordpress/marketing+digital y las siguientes páginas (pg-2, pg-3, etc.) hasta un máximo de 5 páginas. Las URLs se guardan en el archivo plantillas_urls.txt.

2. preview.py

Este script accede a cada URL del archivo plantillas_urls.txt y obtiene información de cada plantilla como el nombre, el enlace de la plantilla y el enlace de vista previa. Los datos obtenidos se guardan en el archivo plantillas_completas.txt.

3. limpio.py

Este script extrae solo los enlaces de vista previa desde plantillas_completas.txt y guarda los enlaces limpios en el archivo urls_limpias.txt.

Uso

Para utilizar los scripts:

  1. Ejecuta urls.py para obtener las URLs de las plantillas.
  2. Ejecuta preview.py para extraer la información de cada plantilla.
  3. Ejecuta limpio.py para obtener los enlaces de vista previa limpios.

Requisitos

  • Python 3.x
  • Instalación de las librerías requests y beautifulsoup4.
  • Acceso a Internet para obtener datos desde Envato Elements.
